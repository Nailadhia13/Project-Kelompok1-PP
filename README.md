* Studi kasus
  Di era modern saat ini, banyak sekali anak kost yang kesulitan mengatur keuangan harian mereka. Tanpa pencatatan yang jelas, sering kali uang bulanan cepat habis tanpa tahu ke mana saja digunakan. Maka dari itu, program ini kami buat agar dapat mempermudah setiap anak kost dalam mencatat pemasukan, pengeluaran, serta memahami kondisi keuangan hariannya. Dengan adanya program ini, diharapkan pengguna dapat lebih bijak dalam mengelola uang sehingga kebutuhan bulanan dapat terpenuhi dengan baik.

Program ini kami beri judul: [BUDGETBUDDY : Kost Daily Tracker]

Di program ini kami menyediakan fitur-fitur sebagai berikut:

1. Muat data untuk membuka kembali catatan keuangan sebelumnya.
2. Tambah pemasukan untuk mencatat uang yang diterima pengguna.
3. Catat pengeluaran agar pengguna dapat merekam setiap transaksi harian.
4. Ringkasan keuangan untuk melihat kondisi pemasukan, pengeluaran, dan sisa saldo.
5. Daftar pengeluaran yang menampilkan detail seluruh aktivitas belanja pengguna.
6. Simpan data agar seluruh catatan keuangan aman dan dapat digunakan kembali di waktu berikutnya.

inti alur mekanisme dari program :
1. program buat file keuangan.txt  (di fungsi simpanData)
2. program baca data dari file untuk update ke program (di fungsi muatData) 
3. user pilih menu fitur 1-4 (di fungsi dari tambah pemasukan sampai ringkasan keuangan
4. sampai akhirnya user milih fitur nomor 5 yg artinya simpan dan keluar 
5. semua data yg udah diinput user akan ditulis ke file dan bakal ngeupdate variabel global yg mana bakal balik lagi mekanismenya ke nomor 2
6. selesai
